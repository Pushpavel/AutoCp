AutoCp

AutoCp Plugin

An Open Source Competitive Programming Plugin for Intellij-Based IDEs

Get Started Introduction

Productive

With an IDE optimized for Competitive Programming, you can start coding in little to no time.

Multi-Language

Supports almost any programming language, build tool and Intellij-Based IDE.

File Templates

Provides simple file templates out of the box which can be customized to your liking.

Flexible

No strict file structure. Testcases to the time constraints are customizable for each problem.

Tests UI

AutoCp uses the built in Tests UI of the IDE to display results.
